应用示例：
1.(动态)公网IP且80端口正常环境，发布网站、做网站服务
（1）使用动态域名解析。

2.(动态)公网IP且80端口被封环境，发布网站、做网站服务
（1）使用80端口映射。
（2）配置本地公网IP加速。

（1）使用80端口映射+网站加速。
4. 域名解析管理。
（1）A记录/cname记录/NS记录/MX记录/URL显性转发/URL隐性转发。

（1）办公应用。OA，SVN……
（2）主机设备。WINDOWS远程桌面，SSH访问LINUX，路由器……
（3）数据库。ORACLE,SQLSERVER,MYSQL……
（4）FTP。
6. 外网访问内网多端口、固定端口、C/S架构、UDP等应用。
（1）使用全端口映射。访问端启用访问者。
（2）适合FTP、管家婆、金碟、视频监控、VPN等所有应用。

（1）无需公网IP，可上网即可开服联机。
